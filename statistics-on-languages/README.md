## Hamming distance

|      Example 1        |       l1        |        l2       |       l3        |        l4        |
| :-------------------: | :-------------: | :-------------: | :-------------: | :--------------: | 
| String 1              |       p         |       a         |       p         |        a         |
| String 2              |       p         |       o         |       p         |        o         |
| Distance hamming (=2) |       0         |       1         |       0         |        1         |



|      Example 2          |       l1        |        l2       |       l3        |        l4        |        l4        |        l4        |
